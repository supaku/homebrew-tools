# Supaku Tools

Homebrew tap for [Supaku Labs](https://github.com/supaku) command-line tools.

## Usage

```sh
brew tap supaku/tools
brew install kith
```

## Available casks

| cask | description | source |
|------|-------------|--------|
| **kith** | macOS CLI bridging Apple Contacts and iMessage for terminal users + AI agents | [supaku/kith](https://github.com/supaku/kith) |

## How releases get here

The version and sha256 fields in each cask are rewritten programmatically by the source repo's release workflow on tag push. Hand edits to the rest of the file are welcome via PR.
